# Поле: password

Компонент: `lte3::components.text` з `type="password"` (викликається через `Lte3::password()`)

Опис

Password input використовує той же шаблон `text`, але при рендерингу початкове значення очищується (`$value = ''`) та встановлюється плейсхолдер за замовчуванням (`********`).

Props / attrs
- `name` (string)
- `label`, `help`, `placeholder`, `class`, `readonly`, `disabled`
- `attrs` — додаткові HTML-атрибути

Приклад (з `examples/components.blade.php`):

```blade
{!! Lte3::password('Password') !!}
```

Особливості

- Початкове значення не виводиться у поле з міркувань безпеки; якщо потрібно показати, використовуйте інший механізм або додатковий плейсхолдер.
- Для показу/приховування пароля можна комбінувати з `secret` режимом (див. нижче) або реалізувати додаткові кнопки.
