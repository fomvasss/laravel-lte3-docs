# Компонент: btnModalClose

Компонент: `lte3::components.btnModalClose` (викликається через `Lte3::btnModalClose()`)

Опис

Проста кнопка для закриття модального вікна (`data-dismiss="modal"`). Може приймати CSS класи та атрибути.

Props:
- `title` (string)
- `attrs.class`

Приклад:

```blade
{!! Lte3::btnModalClose('Close') !!}
```
